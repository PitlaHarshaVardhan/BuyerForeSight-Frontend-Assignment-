# Frontend Buyers

A React application built with Vite for managing buyer information.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Preview the production build:
   ```bash
   npm run preview
   ```

## Available Scripts

- `npm run dev` - Starts the development server on http://localhost:5173
- `npm run build` - Builds the app for production
- `npm run lint` - Runs ESLint on the project
- `npm run preview` - Preview the production build locally

## Dependencies

- React 19.2.4
- React Router DOM 7.13.2
- Lucide React 1.6.0

## Development

This project uses Vite as the build tool and includes ESLint for code quality. The application is configured to use React 19 with TypeScript support.

## Assumptions

- Node.js and npm are installed on the system
- Port 5173 is available for development (default Vite port)
- The application will be run in a modern web browser
